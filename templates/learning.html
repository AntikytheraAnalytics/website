Machine Learning

Machine learning fits statistical models on data to accomplish many useful tasks. We love to apply our passion for machine learning and explore ways in which these models can work for you.

Clustering

Clustering refers to techniques which identify closely related groups of samples in the data. This can be extremely useful for identifying large trends in the data. For example, images of handwritten numbers can be seen to cluster by number.

Classification

Classification learns to assign a label to never before seen data points based on training data. This can be useful for identifying that a data point belongs to one of several known categories. For example, classification techniques can be used to determine the species of an Iris flower with high accuracy, using only a few measurements.

Prediction

Prediction can involve several different tasks. You may want to predict trends in time-series data, predict how a user will like a new product based on previous preferences, or predict when the next user interaction with your platform will occur.

Deep Learning

Deep learning is a broad category of machine learning techniques that involves applying many layers of learned transformations to the input to produce highly expressive models.

Deep learning can be used for a wide variety of tasks, including generating English sentences given a thought vector, classifying and segmenting objects in images, and reproducing an image in the style of a particular painter.
